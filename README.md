# Password Generator

A simple password generator web application built using **HTML, CSS, and JavaScript**.  
The app generates two secure random passwords at the click of a button using a predefined set of characters.

This project was created to practice:
- DOM manipulation
- JavaScript functions and event handling
- CSS layout and component styling
- Structuring a small frontend project

---

## 🚀 Features

- Generates two random passwords
- Password length is configurable in JavaScript
- Uses letters, numbers, and special characters
- Clean and minimal UI
- Responsive-centered card layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript
